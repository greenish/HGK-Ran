• Lorenz Visualization 2

    This is a "nicer" or more "displayable" version of my Lorenz 84 Visualisation. Its made for display during the openhouse day at the HGK Basel and had to run by its self and look interessting without input.

    In this sketch one of the variables (v0 - v8) is animated over time. By steadily increasing the value of one variale, the shape changes slowly till it reaches an unstable state, then the value of the variable will be decreased again. It is possible to choose the value that gets animated by clicking on it.

    Notice each value increases or decreases only 0.0001 per frame and what huge difference this small change makes.

    The shape consists of 1000 points that are calculeted every frame by the displayed formula. On the left side the coordinates of each point are diplayed in a list. On hover you can see where the points actual position is inside the shape.
