• Lorenz Visualization 1

    This is a simple visualization of the "Lorenz 84 Attractor" with the possibility to adjust nine variables of the formula, aswell as the number of iterations.

    The Lorenz 84 Atttactor is based on a formula to which you pass the coordinates of a point p(x,y,z). When runing these coordinates through the formula you get the coordinates for a new point p1(x1, y1, z1). Now you can pass p1 to the formulat to get p2 and so on. All the points are connected by a line.

    The Lorenz 84 formula looks like this:

    dx = (v0 * x - y * y - z * z + v1 * v2) * v3
    dy = (-y + x * y - v4 * x * z + v5) * v6
    dz = (-z + v7 * x * y + x * z) * v8

    The variables (v0-v8) are static numbers. These numbers can be adjusted with the sliders in the sketch to get new interessting shapes.
