• About the Library

    pcaCollisionDetection is a library for Processing (processing.org) that allows you to easily handle a large amount of collision tests.

    This library has been developed for the sketch "pcaRan" as part of a project at the HGK Basel, Switzerland. This library is my very first processing library and its been build more out of the interest to learn about building libraries and wrapping up my project to a nice form, than out of the need for a new library. I do not give any garantee of its usefulness or its functionality. If you can use it, feel free to do so and if you want to, tell me about it! I would love to see what you got out of it.

    What it's doing

    Each instance of the CollisionDetection object holds a set of CollisionElements. The set can then be tested against any other CollisionElement.s Every element that inherits from the class CollisionElement, can be added to the CollisionDetection set and will be mapped to a quadrant in a grid system. Elements that are tested against this set, only need to be tested against elements that lay inside one of the surrounding Quadrants of the tested element. This reduces the amount of collision tests drastically. A collision occourse if the action radiuses of the elements are overlapping. If the collison test is positiv the function CollisionElement.collision() is called for each colliding element.

    Installation

    - Download the latest version of the library from Github here.

    - Unpack it to path/to/Sketchbook/libraries/.

    - Start/Restart Processing. Done.

    - Have a look at the examples and the documentation to see how it works.
