• pcaRan

    This final sketch and the "result" of this project brings it all back together, the chaos, my lorenz shapes, and user interaction.

    Because my initial startingpoint had been "ran" or "chaos" I wanted to bring this aspect back into my work. I wanted to create something that allows some interesting userinteraction, integrates my lorenz shapes whith which I've spent so much time and has in some way a relation or a link to the thoughts of the chaos theory.

    I find it fascinating to think about these shapes that evolve with a certain set of variables (v0 - v8) as a tiny moment in an enormous chaos. Inside this very little window of values that are set for v0-9 the chaos produces by accident something that we humans can grap and that can be seen as something nice and beautifull.

    I didn't want to do any infographics to explain my sketch in order for it to work but I wanted to integrate this thought in some intuitive way.

    So I created a chaos that moves and changes and does not allow to grap and focus any forms at all. And like in the real world, not touched the chaos looks still and like one big unit, but when going closer you learn that its actually moving and changing nonstop. The user can now search through this chaos and is able to find the small moment when the chaos suddenly creates a shape, a shape that exists for a while but then vanishes from the users eyes and is not retrievable but by searching again for a new shape.
